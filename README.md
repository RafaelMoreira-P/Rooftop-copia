# Descrição do Projeto 📌

Este projeto consiste na reprodução de um layout de site de restaurante, desenvolvido como atividade escolar com o objetivo de consolidar conhecimentos em desenvolvimento front-end.

A proposta foi recriar, com o máximo de fidelidade possível, a interface de um site real, trabalhando principalmente com HTML5 e CSS3, sem o uso de JavaScript.

Durante o desenvolvimento, foram aplicados conceitos importantes como:

Estruturação semântica de páginas
Posicionamento de elementos (Flexbox e Position)
Estilização avançada com CSS
Organização de layout e espaçamentos
Uso de fontes externas (Google Fonts)
Criação de seções como hero banner, menu, galeria e formulário de contato

Além disso, o projeto também envolveu desafios práticos, como o alinhamento preciso de elementos sobre imagens, criação de componentes laterais fixos e reprodução fiel de detalhes visuais do layout original.

# Objetivo 🎯

O principal objetivo deste projeto foi desenvolver habilidades na construção de interfaces modernas, reforçando a atenção aos detalhes e a capacidade de transformar um design visual em código funcional.

# Observação ⚠️

Este projeto é apenas para fins educacionais, não possuindo fins comerciais. O layout utilizado foi inspirado/copiado de um site existente para fins de estudo e prática.
